Как привязать Discord к Jade Bio (статус + игра + Spotify)
============================================================

1. Включи режим разработчика в Discord:
   Настройки → Расширенные → Режим разработчика → Вкл.

2. Скопируй свой Discord ID:
   ПКМ по своему нику → «Скопировать ID пользователя»
   (длинное число, например 123456789012345678)

3. Подключи Lanyard (бесплатный сервис статуса):
   • Зайди на https://discord.gg/lanyard
   • Напиши боту в ЛС: .help
   • Следуй инструкции бота (обычно нужно один раз авторизоваться)

4. В кабинете Jade Bio:
   • Вставь Discord ID в поле «Discord ID»
   • Вставь ссылку на профиль: https://discord.com/users/ТВОЙ_ID
   • Сохрани и открой своё био

5. Что будет на странице:
   • Аватар и ник из Discord
   • Статус (онлайн / не беспокоить / и т.д.)
   • Игра или Spotify, если включено в Discord
   • Кнопка «Add on Discord»

Важно:
- Без Lanyard виджет покажет заглушку «Подключи Lanyard».
- Статус обновляется через API https://api.lanyard.rest (уже встроено в сайт).
- Музыка с MP3 в кабинете — отдельно от Spotify в Discord.
